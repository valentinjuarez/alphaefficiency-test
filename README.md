# alphaefficiency-test

Landing page simple creada para simular un sitio promocional de un centro médico especializado en pérdida de peso.

## Descripción

El proyecto está hecho con HTML, CSS y JavaScript puro. La página presenta una sección principal con llamada a la acción, estadísticas, información sobre la consulta, beneficios, preguntas frecuentes y un pie de página con datos de contacto.

## Estructura

- `index.html`: contenido principal de la landing.
- `css/style.css`: estilos visuales y responsive design.
- `js/menu-hamburguesa.js`: comportamiento del menú móvil.
- `js/countdown.js`: cuenta regresiva de la oferta.
- `js/faq-accordion.js`: acordeón de preguntas frecuentes.
- `assets/`: imágenes e íconos usados en la página.

## Funcionalidades

- Menú hamburguesa para pantallas pequeñas.
- Cuenta regresiva en la sección principal.
- Preguntas frecuentes desplegables.
- Diseño adaptable para escritorio y móvil.

## Cómo verla

Solo abre `index.html` en el navegador para visualizar la landing.

## Notas

El contenido está escrito en inglés porque la maqueta original está pensada para un público de Chicago y un centro médico en ese mercado.
